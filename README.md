# Portelio

A mobile app for iPhone that connects two people through a shared widget.

Each user can send one photo per day that appears directly on their partner's home screen widget — no notification, just the quiet pleasure of discovering a new moment someone chose to share with you.

Built for couples, friends, and family.

---

**Stack:** Nuxt 3 · Capacitor iOS · Swift/WidgetKit · Supabase · Tailwind CSS
